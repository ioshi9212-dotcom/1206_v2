# Ray + Jun Knowledge Patch

## Scope

Точечный патч для постоянных знаний Рэя и Джуна + динамических стартовых незнаний.

## Главная логика

- Персонаж может знать факт, но не обязан его раскрывать.
- Закрытое знание не должно превращаться в спойлер через повествование.
- Незнание не делает персонажа тупым: он может предполагать, проверять, врать, притворяться, молчать, давить и ошибаться.
- Догадка не становится фактом без сценического источника.

## Files

- `characters/_templates/knowledge_template.yaml` — общий шаблон knowledge.yaml.
- `characters/ray/knowledge.yaml` — постоянные знания Рэя.
- `characters/jun/knowledge.yaml` — постоянные знания Джуна.
- `state/character_knowledge/ray.json` — динамические стартовые незнания/выводы Рэя.
- `state/character_knowledge/jun.json` — динамические стартовые незнания/выводы Джуна.
- `engine/character_knowledge_runtime_rules.md` — правило применения знаний/незнаний.
- `state/knowledge_state.json` — ссылка на новый rule-файл и runtime policy.
- `characters/jun/jun_knowledge_connections.yaml` — cleaned compatibility file.
- `characters/ray/character.yaml` — минимальная правка конфликтной фразы про память.
- `characters/jun/jun_hidden_past.yaml` — минимальная правка, чтобы не писать, что Джун знает механизм самоблокировки.
- `characters/jun/past.yaml` — cleanup старых маршрутных формулировок, чтобы не цеплять неактуальные действия.
