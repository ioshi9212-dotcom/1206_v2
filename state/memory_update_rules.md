# Memory update rules — 1206 v2

После сцены сохранять только значимые изменения:

- кто что узнал;
- отношения;
- репутацию;
- слухи;
- предметы;
- смену локации/времени;
- продвижение календаря;
- сюжетные обязательства.

Backend не выводит изменения из прозы автоматически: отправляй payload в `apply-turn-result`.
