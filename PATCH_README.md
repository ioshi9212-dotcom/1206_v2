# 1206 v2 — scene rules patch v1

Патч не трогает рабочую API-цепочку напрямую. Он добавляет механику сцены по принципу рабочей Академии, но без академического сюжета/лора.

## Что внутри

1. `gpt/scene_format.md` — технический формат сцены: шапка, тело, граница действия игрока, незакрытый крючок, нижние блоки.
2. `gpt/locks/runtime_scene_rules_digest.md` — короткий обязательный digest правил для runtime.
3. `gpt/locks/calendar_usage_lock.md` — календарь как крючки дня, не готовый сценарий; без привязки к Академии.
4. `gpt/locks/npc_living_scene_rules.md` — generic NPC rules для 1206: фоновые NPC, важные NPC, canon boundary, knowledge boundary.
5. `state/akira_progress_state.json` — базовая панель состояния по нулям.
6. `state/relationship_score_panel.json` — базовая панель отношений по нулям и visible_label.
7. `app/response_size_guard_runtime_patch.py` — обновлённый size-guard contract: 5 нижних блоков, progress files, relationship panel total score, visible labels.
8. `calendar/days/1206-08-31.scene_patch.yaml` — локальный блок только для 31 августа.
9. `tools/apply_1206_0831_calendar_patch.py` — безопасный скрипт, который вставляет блок 31 августа в текущий `calendar/days/1206-08-31.yaml`, не перезаписывая весь файл.

## Как применять

### Основные файлы
Распаковать zip в корень репозитория `1206_v2` с заменой файлов.

### 31 августа
Файл `calendar/days/1206-08-31.yaml` специально НЕ перезаписан целиком, чтобы не потерять большой текущий календарный день.

После распаковки запусти из корня репозитория:

```bash
python tools/apply_1206_0831_calendar_patch.py
```

Или вручную вставь содержимое `calendar/days/1206-08-31.scene_patch.yaml` в `calendar/days/1206-08-31.yaml` перед `active_characters:`.

## Проверка после загрузки

- `начнем` генерирует сцену по каркасу, не как случайный текст.
- В доме Джуна 31 августа нет случайных NPC.
- На базе фоновые рейдеры/дежурные/медперсонал допустимы как среда.
- Эмма/Ирэй до знакомства выводятся по visible label, не внутренними именами.
- Нижний блок содержит 5 разделов: действия, возможные реплики, мысли, состояние, отношения.
- Если NPC обращается к Акире, сцена заканчивается выбором, а не решает за неё.
