# Character File Loading Policy

## Main profile

`*_main_profile.yaml` загружается всегда, если персонаж участвует в сцене.

Содержит:
- имя;
- базовые обращения;
- внешность;
- тело;
- голос;
- текущий характер;
- активные привычки;
- базовый тип энергии;
- поведенческие запреты.

Не содержит полного скрытого прошлого и не раскрывает лор игроку.

## Hidden past

`*_hidden_past.yaml` загружается только если сцена требует:
- прошлое;
- память;
- травмы;
- скрытые связи;
- происхождение;
- тайные причины текущего состояния;
- Эхо / кайросы / Самуэль / кольца / шрамы / дети / смерть / пропажи.

## Knowledge connections

`*_knowledge_connections.yaml` загружается, когда:
- в сцене есть несколько важных персонажей;
- нужно проверить, кто что знает;
- есть риск раскрыть чужую информацию не тем персонажем;
- сцена касается отношений, прошлого, вины, памяти или скрытого лора.

## Thought triggers

`*_thought_triggers.yaml` можно грузить вместе с main, если персонаж является POV или важным фокусом сцены.

Этот файл не раскрывает правду напрямую. Он даёт короткие мысли-улики:
- след кольца;
- шрам;
- вода;
- бесшумный шаг;
- нож;
- окно;
- тепло;
- холод;
- шум;
- чужой голос.

## Запрет

Не использовать hidden-файл для авторского монолога.  
Скрытые факты должны проявляться через сцены, реакции, мысли-улики и чужие знания.
