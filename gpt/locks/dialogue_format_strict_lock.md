# Dialogue format strict lock — 1206 v2

Correct format:

```txt
**Имя/видимый дескриптор** — Реплика. (*короткая ремарка*)
```

Description format:

```txt
*Описание отдельной строкой курсивом.*
```

Rules:

- Speaker name is bold.
- Use long dash after speaker.
- Stage remark inside dialogue is short and italic.
- Do not use names Akira has not heard/read yet; use a visible descriptor.
- Do not give Akira new spoken lines unless the player wrote them outside parentheses.
