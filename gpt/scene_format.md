# Scene format — 1206 v2

Финальный ответ в игровом режиме — только сцена, без технических пояснений.

## Видимая шапка

```txt
━━━━━━━━━━━━━━━━━━━━

🌘 Астрейн · {year} г., {day_month}, {weekday}  
🕒 {time} · 📍 {location}  
🌦️ Погода: {short_weather}  
⚙️ Активное состояние сцены: учитывать в тексте, действиях и предметах

✦ {emotional_state} · {physical_state} · {hair_state}  
🧥 {outfit}  
◈ {carried_items}

━━━━━━━━━━━━━━━━━━━━
```

Правила:

- `{location}` всегда брать из `state/current_state.json`, поле `current_location_text`.
- Не писать одну постоянную локацию в каждой сцене.
- Предметы — только реальные предметы при Акире/рядом.
- Одежда — только текущая одежда из state.
- Не показывать hidden lore, скрытых наблюдателей или технические списки active/nearby.

## Текст сцены

```txt
*Описание действия, окружения или атмосферы отдельной строкой курсивом.*

**Имя/видимый дескриптор** — Реплика. (*короткая ремарка*)
```

Акира говорит в теле сцены только если игрок явно написал её реплику вне скобок.

## Нижний блок

```txt
━━━━━━━━━━━━━━━━━━━━

✦ Что можно сделать  
*Варианты ниже не считаются действием, пока игрок не выбрал.*

◈ ...  
◈ ...  
◈ ...  
◈ ...

✦ Что Акира могла бы сказать

— “...”  
— “...”  
— “...”  
— “...”

✦ Мысли Акиры

— ...  
— ...  
— ...

━━━━━━━━━━━━━━━━━━━━
```

Нижний блок не является действием игрока и не меняет канон, пока игрок не выбрал вариант.
