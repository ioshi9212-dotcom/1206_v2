# canon_lore

Компактная структура лора для 1206 v2.

## Активный файл

- `world.md` — общий лор мира, кайросов, энергии, Эхо, Восточного сектора и безопасных hidden-ограничений.

## Закрытый файл

- `absolute_hidden.md` — авторская правда дальнего канона. Не грузить фоном. Не использовать в сценах. Не раскрывать персонажам.

## Принцип

Не плодить много файлов. Один основной лор-файл с хэштегами и отдельный absolute hidden для того, что не должно всплывать вообще.
